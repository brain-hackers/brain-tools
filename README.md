CE Tools
====================

*開発は終了しました (2011/11)。*

Brain CE 用の各種ツールです。

EdCloser
--------------------

Brain の辞書アプリを終了します。すでに終了している場合は起動します。

KN Memo Pad
--------------------

簡易メモアプリです。ファイルの保存／読み込み、文字列の検索、行折り返し、フォント設定の機能があります。
また、選択した単語の意味を英和辞書で調べることもできます。

USB Storage
--------------------

Brain と PC を USB 接続した状態で、PC から Brain の本体メモリや SD カードにアクセスすることを可能にします。
